Bij Site kan je verschillende onderdelen aanpassen.


Het Site gedeelte

Title

De titel van de website is zichtbaar in de kop van de browser en wordt zichtbaar in Google zoekresultaten.

Author

De auteur van de site. Deze informatie verschijnt niet op de website maar kan worden gebruikt door zoekmachines.

Url

Het webadres van de website. Deze komt ook terecht in de link-lijst in de editor.

Email

Het emailadres van de beheerder van de website. Dit emailadres komt ook terecht in de links lijst die beschikbaar is in de editor.

N.B. Als er formulieren staan op de site die emails versturen dan wordt dit emailadres gebruikt als afzendadres.

Description

Hier voer je een korte omschrijving van de website in. De omschrijving is niet op de website zelf te zien, maar wordt gebruikt door zoekmachines zoals Google. Ook kan Google deze teksten gebruiken als omschrijvende tekst bij zoekresultaten.

Standaard wordt aan elke pagina van de website de tekst die hier staat toegevoegd. Voor een afzonderlijke pagina kan je, indien nodig, een aparte tekst schrijven. De standaard tekst wordt dan niet toegevoegd.

Keywords

Keywords zijn zoektermen. Deze termen zijn niet terug te zien op de website, maar worden door zoekmachines gebruikt. Het is ook mogelijk voor afzonderlijke pagina's zoektermen toe te voegen.

Google analytics

Als je uitgebreidere statistieken wilt dan de FlexyAdmin standaard biedt dan kan je bijvoorbeeld Google Analytics gebruiken. De code daarvan kan je hier invoeren en de website is er meteen klaar voor.
