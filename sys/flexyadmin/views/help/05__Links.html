Links verbinden een pagina met andere webpagina's; bezoekers kunnen via de links die je maakt doorklikken naar andere webpagina’s. Je kunt een link maken naar: - Een andere pagina van je website - Een andere website op internet - Een e-mailadres - Een downloadbaar bestand Een link plaatsen ================= 1. Selecteer de tekst of afbeelding die een link moet worden 2. Klik op het link-icoontje boven de pagina: 3. Er verschijnt een popup waarin je de link kunt typen of kiezen uit een lijst Links toevoegen aan de Link lijst ================================= Je kunt zelf links toevoegen bij 'Links' in het linkermenu.
