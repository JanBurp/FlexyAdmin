Links verbinden een pagina met andere webpagina's; bezoekers kunnen via de links die je maakt doorklikken naar andere webpagina’s. Je kunt een link maken naar: - Een andere pagina van je website - Een andere website op internet - Een e-mailadres - Een downloadbaar bestand Een link plaatsen ================= Om een link op een pagina in je website te plaatsen doe je het volgende: 1. Selecteer de tekst of afbeelding die een link moet worden 2. Klik op het link-icoontje boven de pagina: 3. Maak de link met het pop-upvenster wat nu verschijnt: Op dat pop-upvenster staan deze velden: - **Link URL**: Typ hier je link, bijvoorbeeld: "www.voorbeeldsite.nl", of voor een e-mail: "mailto:mijn@emailadres.nl" - **Link lijst**: Of kies hier je link uit de lijst van links en downloads die FlexyAdmin voor je bijhoudt. (zie hieronder) - **Doel**: Kies hier **Link in een nieuw venster openen** als de link naar een andere site verwijst. ###Link lijst In deze lijst kun je kiezen uit de links die FlexyAdmin voor je bijhoudt of die je zelf daarin hebt geplaatst. De lijst is verdeeld in kopjes: - **Info**: hier staan de links die je hebt ingevoerd bij **Site**, zoals je e-mailadres. - **Site links**: hier staan alle pagina's en subpagina's van je website. Daarmee verwijs je dus naar andere delen van je website. - **Links**: hier staan alle links die je hebt geplaatst bij **Links** (zie hieronder). - **Downloads**: hier staan de bestanden die je hebt geüpload bij **Downloads** (zie hier voor meer uitleg). ###Links toevoegen aan de Link lijst 1. Klik op **Links** 2. Klik linksboven op het icoontje met het plusje 3. Vul de velden in: de **URL** is de link die je wilt toevoegen.
