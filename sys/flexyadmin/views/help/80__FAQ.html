##FAQ: Frequently Asked Questions ###Wat is een browser? Een browser is het programma van je computer (of tablet, of smartphone) waarmee je internetpagina's bekijkt. De meest bekende browsers: - Internet Explorer (alleen voor Windows computers) - Google Chrome - Firefox - Safari ###Welke browser heb ik? - Als je een Windows computer gebruikt, dan gebruik je waarschijnlijk Internet Explorer. - Als je een Apple computer gebruikt, dan gebruik je waarschijnlijk Safari. ###Welke browser is goed? Als je privacy en snelheid belangrijk vindt kies dan voor Firefox. Als je privacy minder belangrijk vindt dan snelheid, dan is Google Chrome ook een goed optie. ###Wat is een zoekmachine? Een zoekmachine is een website waarmee je op internet kunt zoeken naar andere websites. Google is de meest bekende. Ander bekende zoekmachines zijn Bing en Yahoo!. ###Mijn site is niet vindbaar op Google! Oorzaak daarvan is bijvoorbeeld: - Je gebruikt zoektermen die heel generiek zijn, bijvoorbeeld "timmerman" of "vakantie". - Je site is nog maar erg kort online. Het duurt vaak enkele weken tot maanden voordat Google je site vindt. En dan vaak niet op de eerste pagina. - Je site is niet bekend genoeg. Tips om je site op Google beter vindbaar te maken lees je hier. ###De tekst op de site ziet er niet goed uit / het lettertype klopt niet! Waarschijnlijk heb je de tekst op deze pagina niet op de juiste manier vanuit Word geplakt. Lees bij Tekst plaatsen hoe dit moet. ###Als ik op RETURN druk komt er een lege regel, dat wil ik niet Als je op **RETURN** drukt begint er altijd een nieuwe alinea. Als je alleen op een volgende regel verder wilt, druk dan tegelijk op **SHIFT** en **RETURN**. ###Wat betekent: "Het bestandstype welke u probeert te uploaden is niet toegestaan."
