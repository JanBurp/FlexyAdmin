##Verklarende woordenlijst ###Browser Een browser is het programma van je computer (of tablet, of smartphone) waarmee je internetpagina's bekijkt. De meest bekende browsers: - Internet Explorer (alleen voor Windows computers) - Google Chrome - Firefox - Safari ####Welke browser heb ik? Soms wil je webbouwer weten welke browser je gebruikt. Als je niet weet welke browser je gebruikt: - Windows computer, dan gebruik je waarschijnlijk Internet Explorer. - Apple computer, dan gebruik je waarschijnlijk Safari. - Linux computer, weet je het echt niet? ####Welke browser is goed? Als je privacy, snelheid, en mogelijkheden allemaal belangrijk vindt kies dan voor Firefox. ###CMS Is de afkorting voor Content Management System. Een systeem om de inhoud van een website aan te passen. FlexyAdmin is een CMS. ###Zoekmachine Een zoekmachine is een website waarmee je op internet kunt zoeken naar andere websites. Google is de meest bekende. Ander bekend zoekmachines zijn Bing en Yahoo!. ##FAQ ###Mijn site is niet vindbaar op Google! Oorzaak daarvan is bijvoorbeeld: - Je gebruikt zoektermen die heel generiek zijn, bijvoorbeeld "timmerman" of "vakantie". - Je site is nog maar erg kort online. Het duurt vaak enkele weken tot maanden voordat Google je site vindt. En dan vaak niet op de eerste pagina. - Je site is niet bekend genoeg. Tips om je site op Google beter vindbaar te maken lees je hier. ###De tekst op de site ziet er niet goed uit / het lettertype klopt niet! Waarschijnlijk heb je de tekst op deze pagina niet op de juiste manier vanuit Word geplakt. Lees bij Tekst plaatsen hoe dit moet. ###Als ik op RETURN druk komt er een lege regel, dat wil ik niet Als je op **RETURN** drukt begint er altijd een nieuwe alinea. Als je alleen op een volgende regel verder wilt, druk dan tegelijk op **SHIFT** en **RETURN**. ##Foutmeldingen ###"Het bestandstype welke u probeert te uploaden is niet toegestaan." De oorzaak daarvan kan zijn: - als je bij **Foto's** een bestand probeert te uploaden wat geen foto bestand is. - als je bij **Foto's** een foto probeert te uploaden van een onbekend type - als je bij **Downloads** een bestand probeert te uploaden dat niet is toegestaan Oplossing: Als je zeker bent dat het je de foto/bestand wel zou moeten kunnen toevoegen, neem dan contact op met de bouwer van je site. Die kan de instellingen veranderen.
