FlexyAdmin is onder andere gebaseerd op het Open Source PHP framework CodeIgniter.

Als je meer uit FlexyAdmin wilt halen is het slim om de handleiding van CodeIgniter door te lezen. Hieronder enkele richtlijnen voor een minimaal begrip van CodeIgniter.

Wat je in ieder geval moet lezen van CodeIgniter

  • Table of Contents -  een overzicht
  • Supported Features - om een idee te hebben van de mogelijkheden
  • Application Flow Chart - hoe werkt CodeIgniter globaal
  • Model-View-Controller - over het MVC model
  • Views - Views
  • Using CodeIgniter libraries - hoe je CodeIgniter libraries laad

En in ieder geval deze CodeIgniter Classes

  • URI Class - hoe haal je de huidige uri op
  • Database Class - overzicht van de Database Class
  • Active Record Class - onderdeel van de Database Class waar FlexyAdmin veel gebruik van maakt en eigen uitbreidingen op heeft.

Wat zijn belangrijke verschillen tussen FlexyAdmin en CodeIgniter

  • CodeIgniter System map is: sys/codeigniter
  • CodeIgniter Application map is: sys/flexyadmin 
  • Aanpassingen aan index.php (o.a. herkenning van localhost, toevoegen SITEPATH)
  • standaard uri-routing is aangepast zodat alle uri's naar de frontend controller gaan: site/controller.php. Behalve uri's beginnend met: admin, rss en file.
  • Diverse aanpassingen en extra Classes / Helpers
