Javascript bestanden voor je site vind je in site/assets/js.

Standaard vind je daar al het bestand site.js. Dat kun je gebruiken als start voor de javascript van jouw site. Het is een jQuery template.

jQuery 

jQuery is een zeer handige javascript library waarmee je eenvoudig browser-onafhankelijke scripts kunt maken. Voor effecten, DOM-manipulatie, AJAX etc.

FlexyAdmin wordt standaard geïnstalleerd met jQuery en enkele jQuery plugins.
