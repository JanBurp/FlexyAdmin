where()

where() is geen nieuwe functie van de Database class maar heeft enkele uitbreidingen die het zoeken in relaties vergemakkelijken.

Zoeken in veel op één relaties

Zoeken in veel op veel relaties

Het resultaat van onderstaand voorbeeld is een array van tbl_menu met alleen rijen die gekoppeld zijn via een veel op veel relatie met tbl_links waarvan tbl_links.str_title = 'FlexyAdmin'.

$this->add_many();
$this->db->where( 'rel_menu__links.str_title', 'FlexyAdmin' );
$result = $this->db->get_result( 'tbl_menu' );
