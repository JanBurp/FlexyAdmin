Uitbreiding van de standaard CodeIgniter helper: string_helper. Wordt standaard geladen.

Functies om delen van strings te selecteren

get_prefix()

(string) get_prefix( (string) $string [, (string) $split='_' ] ) 

Pakt het eerste deel van een string nadat hij de string heeft verdeeld in delen aan de hand van een scheidingskarakter. Standaard is het scheidingskarakter een underscore '_'.

echo get_prefix( 'str_example' );
echo get_prefix( 'tbl_example.id', '.' ); 

Geeft als resultaat:

str
tbl_example 

get_suffix()

(string) get_suffix( (string) $string [, (string) $split='_' ] ) 

Zelfde als get_prefix() maar dan het laatste deel van de string verdeeld door een scheidingskarakter.

echo get_suffix( 'str_example' );
echo get_suffix( 'home/pagina/sub_pagina', '/' ); 

Geeft als resultaat:

example
home 

remove_prefix()

(string) remove_prefix( (string) $string [, (string) $split='_' ] ) 

Verwijderd eerste deel van een string.

echo remove_prefix( 'str_example' );
echo remove_prefix( 'tbl_example.id', '.' ); 

Geeft als resultaat:

example
id 

remove_suffix()

(string) remove_suffix( (string) $string [, (string) $split = '_' ] ) 

Verwijderd laatste deel van een string.

echo remove_suffix( 'str_example' );
echo remove_suffix( 'home/pagina/sub_pagina', '/' ); 

Geeft als resultaat:

str
home/pagina
