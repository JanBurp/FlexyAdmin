Met het linksymbool kan je van een geselecteerde tekst een link maken. Dit kan een link zijn naar andere pagina of subpagina van je website, of naar een andere website op internet.

Om een link te maken selecteer je het gedeelte van de tekst dat de link moet worden. De linksymbolen worden nu actief.
N.B. Door een afbeelding te selecteren kan je ook een link aan een afbeelding toevoegen!

De tekst is geselecteerd, de link symbolen zijn nu actief

Wanneer je nu op het link symbool klikt krijg je het venster Link invoegen/bewerken te zien. Als je op de linklijst klikt zie je alle links binnen je website.

Het links dropdownmenu

Onder info staan de links die zijn ingevoerd onder Site, zoals het standaard e-mail adres.

In het rijtje site links staan alle pagina's en subpagina's van de website. Je kan zo naar andere delen van de website verwijzen.

Onder downloads staan de bestanden die je hebt geüpload in het download gedeelte.


Het Doel dropdownmenu

Wanneer je op Doel klikt krijg je een dropdownmenu waarin je kan kiezen of je de link in hetzelfde venster wil laten openen, of in een nieuw venster.

Het Klasse dropdownmenu

Bij Klasse kan je aangeven hoe de link er uit moet zien.
De link kan bijvoorbeeld een grijs kader krijgen, of een rand er omheen. Ook kan de link er als gewone tekst uitzien en alleen actief worden wanneer je er met de cursor overheen gaat.

Zo ziet het er op de website uit: wanneer je met de muis over de tekst gaat wordt de link actief

Om meer links toe te voegen naar andere websites op het internet ga je naar Links in de lijst links in het scherm.

In de lijst Links zie je welke links er zijn toegevoegd

Om een nieuwe website aan de lijst toe te voegen klik je op het plusje linksboven. Je krijgt het volgende venster te zien:

Het links venster

Bij Title geef je de link een naam. In het vak Url typ je het adres van de website. Begin altijd met http:// en typ daar aanvast het webadres. In dit geval, http://www.voorbeeldsite.nl

De link is nu toegevoegd aan de linklijst en kan nu gemakkelijk gebruikt worden in de tekst.

Module links toevoegen

Wanneer je bij het maken of wijzigen van een  pagina de module Links gebruikt verschijnt de gehele Links-lijst.
