Afbeeldingen bestaan er in 2 soorten:

  • Afbeeldingen die nodig zijn voor de layout van de site
  • Afbeeldingen die de gebruiker op de site kan plaatsen

Afbeeldingen voor de layout

Deze vind je in de map site/assets/img. Deze zul je er zelf in zetten voor gebruik met de stylesheets en de views.

Afbeeldingen van de gebruiker

Deze komen standaard in de map site/assets/pictures. De gebruiker kan hier zelf bestanden naar uploaden. Deze komen standaard terecht in de lijst die vanuit de editor gebruikt kan worden om afbeeldingen in de tekst te plaatsen.

Bij Img Info kun je instellen of tijdens het uploaden de afbeeldingen geresized moeten worden en of er eventueel meerdere formaten moeten worden gemaakt.

Je kunt zelf meerdere mappen aanmaken in site/assets om diverse afbeeldingen te groeperen en diverse formaten te kunnen maken. Bij Media Info en Img Info stel je dat verder in.

Andere bestanden of meer mappen voor Afbeeldingen

Je kunt in site/assets ook mappen maken voor andere bestanden dan afbeeldingen. Bijvoorbeeld pdf bestanden (zie ook bij links) of flash animaties. Standaard bestaat al de map Downloads waar bijvoorbeeld pdf bestanden in kunnen.

Een nieuwe map aanmaken

Je maakt daarvoor eerst de echte map aan in site/assets. Daarna ga je naar Media Info en maakt daar een nieuw item aan. Je vult daar in ieder geval hetvolgende in:

  • UI Name - dit is de menu naam zoals de map zichtbaar wordt in FlexyAdmin
  • Path - Kies hier de map die je net hebt aangemaakt.
  • Types - de extensies van de bestanden die in deze map mogen staan. Alle andere bestanden worden gewijgerd.
  • Zet In Img List aan als het afbeeldingen zijn en je de gebruiker de mogelijkheid wilt geven deze afbeeldingen toe te kunnen voegen in de HTML editor.
  • Zet In Links List aan als je deze bestanden in de link lijst terecht wilt laten komen. Zie bij links.
