De gebruiker kan op verschillende maniere links in een tekst maken. Gewoon de url direct intypen, of de url kiezen uit de link lijst.

Link lijst

De linkt lijst is een bestandje in de map site/assets/lists. De links in dat bestand komen uit diverse bronnen:

  • De links die de gebruiker heeft ingevoerd bij Links
  • De url en het email adres dat de gebruiker heeft ingevuld bij Site
  • Alle interne links van de pagina's aangemaakt bij Menu. (Of bij complexere sites uit het samengestelde Menu Result.)
  • Links naar alle downloadbare bestanden die de gebruiker bij Downloads kan uploaden.

Meer links

Als je net als bij Downloads andere bestanden wilt toevoegen in de link lijst kun je bij Media Info het vinkje In Link List aanvinken. Je kunt, net als voor Afbeeldingen geld, ook meerdere mappen maken met downloads.
