Snelle installatie

  • Maak een map voor je site.
  • Pak FlexyAdmin uit in die map.
  • Maak een nieuwe MySql database aan.
  • Vul de juiste gegevens van je database in: site/config/database(_local).php.
  • Ga met je browser naar de site. Als alles goed is ingesteld zal FlexyAdmin nu starten met de demo database.
  • Klaar om in te loggen!

Inloggen / nieuwe gebruikers

Standaard bestaan er twee gebruikers:

  • user (wachtwoord: user) - deze kan alleen de inhoud van de site aanpassen.
  • admin (admin) - deze gebruiker kan bij alle instellingen van FlexyAdmin
Gebruikers kun je aanpassen (naam/wachtwoord/rechten) bij Users.
 

Maak je eigen (eenvoudige) aanpassingen

  • Pas HTML aan in de views: site/views/*
  • Je hoofdview waar alles samen komt is: site/views/site.php
  • Pas de stylesheets aan in site/assets/css/*
  • Stop eventuele afbeeldingen voor de styling in de map: site/assets/img
  • Eventuele Javascripts pas je aan in site/assets/js. Zorg dat in site/views/site.php de javascript bestanden worden geladen (staat standaard tussen comments).
  • Maak eventuele meer mappen voor gebruikersbestanden aan in site/assets en bij Media Info.
  • Stel bij Img Info in hoe afbeeldingen geresized worden als de gebruiker een afbeelding upload.

Aanpassingen voor gevorderden

  • Bepaal met site/config/config.php hoe de controller zich gedraagd (zie ####).
  • Maak je eigen velden en tabellen aan in de database.
  • Maak nieuwe modules in site/libraries (zie ####).
  • Gebruik samengestelde menu's (zie ####).
  • Maak je site meertalig (zie ####).
  • Maak eigen plugins voor de backend (zie ####).
  • Voor het maken van eigen modules of plugins: lees de diverse hoofdstukken bij API en verdiep je in de diverse Classes van CodeIgniter.
