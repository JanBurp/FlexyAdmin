Links zijn manieren om jouw website te verbinden met andere webpagina's. Links in je tekst maken ----------------------- Je kunt links maken naar: - Een andere pagina van je website - Een andere website op internet - Een email-adres - Een link van een downloadbaar bestand Om een link te maken doe je hetvolgende: - Selecteer een stuk tekst of een afbeelding dat de link moet worden - Klik op dit icoontje Je krijgt nu een popup-venster te zien waar je de link kunt maken. Je kunt een link met de hand intypen bij **URL** of kiezen uit de lijst bij **Link Lijst**. - Onder info staan de links die zijn ingevoerd bij Site, zoals het standaard email adres. - Onder site links staan alle pagina's en subpagina's van je website. Je kan zo naar andere delen van de website verwijzen. - Onder links staan alle links die je hebt toegevoegd bij **Links**. (zie hieronder bij _Veelgebruikte links_) - Onder downloads staan de bestanden die je hebt geüpload in het download gedeelte. Doel aanpassen -------------- Wanneer je op Doel klikt krijg je een dropdownmenu waarin je kan kiezen of je de link in hetzelfde venster wil laten openen, of in een nieuw venster. Veelgebruikte links ------------------- Bij Links kun je links toevoegen die je later kunt gebruiken in je teksten. Je voegt een link toe door linksboven op het icoontje met het plusje te klikken. NB. Het is niet nodig om http:// toe te voegen, dit gebeurd automatisch.
