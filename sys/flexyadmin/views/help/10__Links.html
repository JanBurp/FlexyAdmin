Wat zijn Links -------------- Links verbinden een pagina met andere webpagina's; bezoekers kunnen via de links die je toevoegt doorklikken naar andere webpagina’s. Je kunt een link maken naar: - Een andere pagina van je website - Een andere website op internet - Een e-mailadres - Een downloadbaar bestand Een link maken in een pagina ---------------------------- Om een link op een pagina in je website in te voegen doe je het volgende: - Ga naar de pagina (via _Menu_) - Selecteer het stukje tekst of de afbeelding die een link moet worden - Klik op het link-icoontje boven de pagina - Maak de link met het pop-upvenster wat nu verschijnt - Op dat popup-venster staan deze velden: - **URL**: hier kun je je link intypen. - **Link lijst**: In plaats van de link intypen bij **URL** kun je een link kiezen uit de lijst van links en downloads die FlexyAdmin voor je bijhoudt. Onder _info_ staan de links die je hebt ingevoerd bij Site, zoals je e-mailadres. Onder _site links_ staan alle pagina's en subpagina's van je website. Daarmee verwijs je dus naar andere delen van je website. Onder _links_ staan alle links die je hebt toegevoegd bij **Links**. Onder _downloads_ staan de bestanden die je hebt geüpload in **Downloads**. - **Doel**: Standaard opent een link in hetzelfde venster en verdwijnt de huidige pagina. Als je hier kiest om een link te openen in een nieuw venster blijft de huidige pagina (van je site) ook open. Kies dat als je naar een andere site linkt. - **Titel**: Dit kun je leeglaten. Een veelgebruikte link toevoegen aan de Link Lijst -------------------------------------------------- FlexyAdmin bewaard automatisch een hele lijst van links die je kunt kiezen bij **Link lijst**: - Je emailadres en link van je eigen site ingevuld bij _Site_ - Alle pagina's van je eigen site - Alle bestanden die je bij **Downloads** hebt toegevoegd - Én alle links die je invult bij **Links** Een link daaraan toevoegen: - Klik op **Links** - Klik linksboven op het icoontje met het plusje - Vul de velden in. De titel kun je eventueel zelf verzinnen, de Url kopieer je. Het is niet nodig om die met http:// te beginnen, www. is voldoende.
