Wat zijn Links -------------- Links verbinden een pagina met andere webpagina's; bezoekers kunnen via de links die je toevoegt doorklikken naar andere webpagina’s. Je kunt een link maken naar: - Een andere pagina van je website - Een andere website op internet - Een e-mailadres - Een downloadbaar bestand Een link maken in een pagina ---------------------------- Om een link op een pagina in je website in te voegen doe je het volgende: - Selecteer het stukje tekst of de afbeelding die een link moet worden - Klik op het link-icoontje boven de pagina: - Maak de link met het pop-upvenster wat nu verschijnt: - Op dat popup-venster staan deze velden: - **URL**: hier kun je je link intypen. - **Link lijst**: In plaats van de link intypen bij **URL** kun je een link kiezen uit de lijst van links en downloads die FlexyAdmin voor je bijhoudt. (zie hieronder) - **Doel**: Standaard opent een link in hetzelfde venster en verdwijnt de huidige pagina. Als je hier kiest om een link te openen in een nieuw venster blijft de huidige pagina (van je site) ook open. Kies dat als je naar een andere site linkt. ###Link lijst In de link lijst kun je kiezen uit de links die FlexyAdmin voor je bijhoudt, de lijst is verdeeld in deze kopjes: - **info** hier staan de links die je hebt ingevoerd bij Site, zoals je e-mailadres. - **site links** hier staan alle pagina's en subpagina's van je website. Daarmee verwijs je dus naar andere delen van je website. - **links** hier staan alle links die je hebt toegevoegd bij **Links** (zie hieronder). - **downloads** staan de bestanden die je hebt geüpload bij **Downloads** (zie hier voor meer uitleg). ###Veelgebruikte links toevoegen aan de Link Lijst - Klik op **Links** - Klik linksboven op het icoontje met het plusje - Vul de velden in. De titel kun je eventueel zelf verzinnen, de URL is de link die je wilt toevoegen. Het is niet nodig om die met http:// te beginnen.
