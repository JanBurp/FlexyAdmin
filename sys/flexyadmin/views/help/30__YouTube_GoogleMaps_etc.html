Je kunt ook YouTube filmpjes, Google Maps of bijvoorbeeld een foto van Flickr toevoegen aan je tekst. - Ga met je cursor naar de plek in de tekst waar het YouTube filmpje moet komen en klik op dit icoontje. - Plak nu in het venster de 'embed' code (ook wel 'insluitcode' genoemd) die je heb gekregen van YouTube etc. Een Youtube filmpje toevoegen ----------------------------- - Ga naar het filmpje op de YouTube site - Klik op **Delen** / **Share** - Klik nu op Embed en je krijgt de code die je nodig hebt om het filmpje op je website in te voegen. - Pas eventueel de groote aan - Kopieer de embed code en plak deze in het venster: Het venster met de Embed code ingevoerd: Klik op Invoegen en de code wordt in de tekst geplakt. Je ziet nu in de tekst het filmpje. Een Google Maps locatie toevoegen --------------------------------- - Ga in Google Maps naar je locatie - Klik op het linksymbooltje (rechtsboven) - Kopieer de embed code en plak deze in het 'embed' venster bij je tekst
