##YouTube en Google Maps Je kunt ook YouTube filmpjes of een locatie uit Google Maps plaatsen. Die gebeurt met een embed (of insluit)-code. Je kunt een embedcode bijvoorbeeld krijgen bij: - een YouTube film - een locatie of routebeschrijving van Google Maps - een foto/album van Flickr of Picasa - muziek van SoundCloud ###Embed code plaatsen 1. Ga met je cursor naar de plek in de tekst waar je de code wilt plaatsen 2. Klik op het Embed icoontje 3. Plak nu in het pop-upvenster de code 4. Klik op **Invoegen** ###Een YouTube filmp plaatsen Hier een voorbeeld van een YouTube film: 1. Ga naar het filmpje op de YouTube site 2. Klik op **Delen / Share** 3. Klik nu op **Embed** / **Insluitcode** en je krijgt de code die je nodig hebt om het filmpje op je website te uploaden: 1. Pas eventueel de grootte aan 2. Kopieer de embed-code en plak die in het pop-upvenster ###Een Google Maps locatie plaatsen Hier een voorbeeld van een Google Maps locatie: 1. Ga in Google Maps naar je locatie 2. Klik op het linksymbooltje (rechtsboven) 3. Kopieer de embed code en plak deze in het pop-upvenster
