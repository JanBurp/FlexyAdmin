Met het groter dan/kleiner dan symbool kan je Youtube filmpjes, een Google maps locatie of bijvoorbeeld een foto van Flickr toevoegen.
Door op het symbooltje te klikken verschijnt er een venster. Hierin kan je de code plakken die je krijgt op Youtube, Google, Flickr, Picasa etc.

Invoegen gaat voor alle sites op dezelfde manier. Hieronder wordt het uitgelegd voor Youtube en Google Maps:

Een Youtube filmpje toevoegen


Een filmpje met uitleg van het FlexyAdmin systeem met de Embed code

Door op Share te klikken krijg je een link naar het filmpje te zien. Klik nu op Embed en je krijgt de code die je nodig hebt om het filmpje op je website in te voegen. Kopieer deze code en plak deze in het venster:

Het venster met de Embed code ingevoerd

Klik op Invoegen en de code wordt in de tekst geplakt. Je ziet nu in de tekst het filmpje.

Een Google Maps locatie toevoegen

Wanneer je in Google Maps de locatie hebt opgezocht zie je rechtsboven drie symbooltjes staan. Wanneer je op het linksymbooltje klikt krijg je de embed code te zien.


Het link symbool

De embed code

Kies nu voor  HTML in website plakken. Plak deze code in het venster. Klik op invoegen en de Google Maps locatie is aan de pagina toegevoegd.
